* Назначение
Этот проект определяет базовые функции для создания и отображения структуры данных типа [[https://ru.wikipedia.org/wiki/Граф_(математика)][Граф]].

Проебразование графа в формат пригодный для визуализации осуществляется при помощи программы [[https://graphviz.org][Graphviz]].
